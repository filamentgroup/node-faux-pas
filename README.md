# node-faux-pas

A command line utility to test a URL for faux web font rendering and mismatched web font code.

## Installation

```
npm install -g node-faux-pas
```

## Usage

```
$> fauxpas MY_URL_HERE
```

## Options

```
  -u, --url url   The url to test.
  -h, --help      Display this help documentation.
```
